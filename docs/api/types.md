# 类型定义

本节记录 ToolRegistry 面向 LLM API 的类型定义。

## 工具调用类型

用于转换不同 provider payload 的标准化工具调用和工具结果模型。

::: toolregistry.llm.tool_calls
    options:
        show_source: false
        show_root_heading: true

## Content Blocks

用于构建工具结果消息的文本和多模态 content block 辅助类型。

::: toolregistry.llm.content_blocks
    options:
        show_source: false
        show_root_heading: true
